# 🧠 EchoMind – Memory Capsule Manager

EchoMind is a modern React-based memory capsule application that helps users preserve meaningful moments, thoughts, and reflections in a beautiful and organized way.

Instead of using a traditional notes app, EchoMind focuses on emotional memories and personal experiences.

---

## ✨ Features

* 🧠 Save meaningful memories
* 📝 Add memory title and reflection
* ❌ Delete memory capsules
* 📅 Auto date display
* ⚛️ Built using React Hooks
* 🔄 Dynamic rendering using map()
* ✅ Form validation
* 🌌 Modern futuristic UI
* 📱 Responsive design

---

## 🛠️ Technologies Used

* React.js
* JavaScript
* CSS3
* Vite

---

## 📚 React Concepts Used

* useState()
* useEffect()
* Dynamic rendering with map()
* Event handling
* Conditional rendering

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone YOUR_GITHUB_LINK
```

2. Open project folder

```bash
cd Assignment-2
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```
